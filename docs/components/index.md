# Components

MorphUI provides a rich set of interactive UI components. All components integrate with the [theme system](../theme/index.md) and update automatically when the theme changes.

---

| Component | Description |
|---|---|
| [Button](button.md) | Text buttons and icon buttons with ripple and hover |
| [Chip](chip.md) | Compact interactive chips for filtering and input |
| [Dialog](dialog.md) | Modal overlays with scrim layer |
| [Divider](divider.md) | Horizontal and vertical visual separators |
| [Dropdown](dropdown.md) | Dropdown menus with optional search filtering |
| [Label](label.md) | Themed labels with icon and auto-sizing support |
| [List](list.md) | Scrollable lists with themed items |
| [Menu](menu.md) | Context and navigation menus |
| [Progress](progress.md) | Linear and circular progress indicators |
| [Selection](selection.md) | Checkboxes, radio buttons, and switches |
| [Text Field](textfield.md) | Text inputs with validation and error states |
| [Tooltip](tooltip.md) | Hover-triggered informational overlays |
